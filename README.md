# Steps to build this projects

```bash
cd tdc-floripa
```

```bash
npm install -g bower
```

```bash
npm install http-server -g
```

```bash
cd [ANY-PROJECT-FOLDER]
```

```bash
bower install
```

```bash
http-server .
```
